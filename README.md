# Internet Banking Application

## Descriere

Aplicatia reprezinta un sistem de tip Internet Banking destinat exclusiv persoanelor fizice. Astfel, utilizatorii isi pot gestiona conturile bancare, pot efectua tranzactii si pot sa isi configureze setari financiare intr-un mod intuitiv.

Sistemul se muleaza pe functionalitatile principale domeniului bancar, precum administrarea conturilor bancare, emiterea cardurilor, efectuarea si programarea tranzactiilor, oferind fluxuri coerente si cat mai apropiate de un sistem real.

## Scopul proiectului

Ne propunem sa dezvoltam o aplicatie web completa care sa modeleze un sistem de Internet Banking la nivel functional, punand accent pe logica de business si pe corelarea dintre componentele sistemului. Aplicatia urmareste sa demonstreze modul in care sunt gestionate operatiuni precum tranzactiile financiare, limitele utilizatorilor, accesul la conturi si automatizarea anumitor procese.

## Functionalitati principale

Aplicatia ofera urmatoarele functionalitati principale:
- creare individ si utilizator
- autentificare in sistem
- creare si administrare conturi bancare
- suport pentru conturi de tip multiaccount
- emitere carduri asociate conturilor
- efectuare tranzactii (plati, transferuri, schimb valutar)
- configurare limite tranzactionale
- clasificare tranzactii prin categorii
- executie tranzactii urgente sau standard
- programare plati pentru executie ulterioara
- procesare automata a platilor programate
